# Full Request Logger

Easy access to full request logs via a web UI.

## Installation

```ruby
# Gemfile
gem 'full_request_logger'
```

## Usage

Access request logs via /rails/conductor/full_request_logger/request_logs/:id where id is the X-Request-Id.
